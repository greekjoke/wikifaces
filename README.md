# Wiki Faces

Приложение для просмотра фотографий различных персон с википедии.

## Disclaimer

Все данные берутся с ресурсов Википедии, поэтому претензии к точности и достоверности информации переадресую авторам материалов указанной платформы. Время загрузки данных может варьироваться в зависимости от загруженности серверов вики.

## Настройки

В меню "настройки" доступны некоторые опции, влияющие на работу всего приложения. Также, там есть кнопка сброса настроек — это может пригодиться, если вдруг что-то внутри приложения "сломается" (хотя, не должно). Опции:

- Показывать подписи к фото. При включении этой опции, под каждым фото будет отображаться имя и возможно, некая дополнительная информация. Кликнув на имя, вы сможете перейти на страницу в вики.

- Поиск лиц. При открытии фотографии, программа будет пытаться искать на ней лицо, чтобы показать его поближе. Этим можно в некоторой степени управлять.

## Коллекции

В коллекции собраны фото людей, объединённых неким общим признаком.

Просмотр осуществляется постранично, причём кол-во фото на странице можно менять - кнопки на панели инструментов "+" и "-". Также, есть кнопки позволяющие управлять сортировкой элементов коллекции - по возрастанию "▲", по убыванию "▼" и "перемешать". Навигация по страницам производится с использованием специальных кнопок на панели инструментов, либо клавиш клавиатуры: стрелки влево/вправо, Home, End.

При первом отображении фото, программа пробует найти на ней лицо. Такой поиск может занимать до десятка секунд. Иногда бывает, что область лица определяется неверно. Тогда вы можете отредактировать ракурс кликнув на фото. В этом режиме можно пользоваться мышкой, в том числе колесом.

## Игры

В этом разделе собраны игры построенные на угадывании каких-то характеристик людей, изображённых на конкретной фотографии. За правильные (с точки зрения приложения) ответы начисляются балы. Также, дополнительные балы начисляются за раунд со всеми правильными ответами.

## Прочее

Кнопка Backspace клавиатуры работает похоже на кнопку Назад браузера, - возвращает на предыдущий раздел приложения. Заголовок приложения можно свайпать для пролистывания.

Сообщение в игре "Получено недостаточно данных" может иногда появляться, это нормально. Бывает, что для запрашиваемой комбинации параметров не хватает данных в определённых странах (выбираются случайно). В этом случае нужно нажать кнопку "Старт" повторно.
